ShopTalk AI – Multi-Agent SQL Analytics Platform

ShopTalk AI is an AI-powered analytics platform that transforms natural language business questions into SQL queries, business insights, and interactive dashboards.

Features

* Natural Language to SQL
* Multi-Agent Architecture
* SQL Query Execution
* Business Insight Summarization
* Interactive Dashboards & Visualizations
* Retail Sales Analysis

Multi-Agent Workflow

1. SQL Generator Agent
2. SQL Executor Agent
3. Answer Summarizer Agent
4. Visualization Agent

Tech Stack

* Python
* MySQL
* Streamlit
* LangGraph
* Plotly
* Pandas
* LLMs

Example Use Cases

* Top-performing products analysis
* Revenue and profit trends
* Sales growth comparison
* Business intelligence dashboards

Installation

pip install -r requirements.txt

Run the Project

streamlit run app.py

Project Goal

Build an AI-powered business intelligence assistant capable of generating insights and dashboards automatically from natural language queries.
